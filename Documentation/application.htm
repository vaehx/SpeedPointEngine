Writing a SpeedPoint Application in Visual Studio

Tested with Visual Studio 2017.

  1. Create an empty Win32 Project
  2. Add 'SP_UNITTEST' to preprocessor definitions
  3. Add 'SpeedPoint\SpeedPointEngine\Source\SpeedPointEngine' path to VC++ Include Directories
  4. Add 'SpeedPoint\Debug' or 'SpeedPoint\Release' to VC++ Library Directories
  5. Add following libraries to Linker > Include:
    1. SpeedPointCore.lib
    2. SpeedPoint3DEngine.lib
    3. SpeedPointDirectX11Renderer.lib
    4. SpeedPointGameFacilities.lib
    5. SpeedPointPhysics.lib
    6. SpeedPointFileFormats.lib
