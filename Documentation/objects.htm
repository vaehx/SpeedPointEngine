Geometry management

Object architecture

struct IObject;
struct IRenderableObject : public IObject, public STransformable;
  • Main Interface: IObject
  • Sub-Interfaces (inherit from IObject): IRenderableObject
  • Implementations: StaticObject, Terrain, (AnimatableObject, VegetationObject, WaterSurface, Ocean, ...)

Existing object implementations

Static Object

class StaticObject : public IRenderableObject;
  • use to render objects, that are not animated, scripted or something else
  • Examples: Rock, Cliff, House, ...
  • Not: Tree, Water, Car, ...
  • Contains single Geometry instance
  • Can be inherited from to create extended implementations of IObject
